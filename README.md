# <img height="25" src="https://raw.githubusercontent.com/marwin1991/profile-technology-icons/refs/heads/main/icons/python.png"> mini-auspex: a Python library for ultrasonic data import, post-processing, and image reconstruction

This repository is a suplementary material to the paper **mini-auspex: a Python library for ultrasonic data import, post-processing and imaging**, presented at the 14th European Conference on Non-Destructive Testing (ECNDT 2026).

The _mini-auspex_ package contains the functions to process and analyze nondestructive ultrasound tests. Among these codes are tools for reading inspection files from various sources, data preprocessing algorithms, image reconstruction algorithms, and codes for presenting results. This repository contains several examples demonstrating the use of the _mini-auspex_ package. All examples are configured to run in the [Google Colab](https://colab.research.google.com/) environment.
